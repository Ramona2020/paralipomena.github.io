We'll be using several technologies in this course. The primary tools are listed below.

ExistDB is an open source XML Database. We'll be using ExistDB as to store, query, and analyze XML documents using XQuery.

GitHub

GitHub is a social network for coders. We'll be using GitHub to host and share our code during the course.

The oXygen XML Editor is a leading integrated development environment (IDE) for XML, XQuery, XSLT and other XML-based standards. We'll be using oXygen as our primary editing and programming environment.

RStudio is an open source integrated development environment (IDE) for the R programming language. We'll be using R to analyze and visualize aspects of our digial texts.
